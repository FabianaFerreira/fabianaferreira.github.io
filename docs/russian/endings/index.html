Prepositional Case

It's simply a grammatical ending that must be added after certain prepositions, which are: в (in), на (on), о (about, concerning), при (attached to, in presence of).

  • Nouns that end with a consonant add -е; nouns ending in -й, -а or -о change the vowel to -е;
  • Nouns which end -ь: if it's masculine, -ь changes to -е, but if it's feminine -ь the changes to -и;
  • Nouns which end -я or -е also have the prepositional ending -е unless the letter before the -я or -е is -и.
